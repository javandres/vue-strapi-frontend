## Back-End Strapi Setup

From [Strapi.io](https://strapi.io/getting-started)

Install Strapi:
```bash
$ npm install strapi@beta -g
```

Create a project:
```bash
$ strapi new myProject
```

Enter project and Launch Strapi server:
```bash
$ cd myProject
$ strapi start
```
